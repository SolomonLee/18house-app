◆ HTML 撰寫風格
	class = {通用名稱}_{層級名稱} {功能名稱} {功能樣式(可省略)}
		e.g.  "list_item menu_list menu_list_style2"
		
	層級名稱 : block > box > item
		層級可包含同層 但不可包含大於之層數


◆ 檔案名稱 以 大駝峰式 命名方式 (自動建立除外)


◆ javascript variable 名稱 以 小駝峰式 命名方式


◆ RWD 切換之視窗大小界線 以 Bootstrap(ver4) 為主
